# Traffic-Signal-Program-
<br>
Traffic Signal Program In Python It shows the color signal as instruction (stop,ready,Go)
<br>
ch = input("Enter traffic light color: ")
<br>
if ch == "Red ":
    print("STOP!!!")
<br>
elif ch == "Green":
     print("GO")
<br>
elif ch == "Orange":
    print("BE READY...")
<br>
else:
    print("Invalid choice")
