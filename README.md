<br>
Traffic Signal Program In Python It shows the color signal as instruction (stop,ready,Go)
<br>
ch = input("Enter traffic light color: ")
if ch == "Red ":
    print("STOP!!!")
elif ch == "Green":
    print("GO")
elif ch == "Orange":
    print("BE READY...")
else:
    print("Invalid choice")
